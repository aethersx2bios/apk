# AetherSX2 APK

AetherSX2 is a PlayStation 2 emulator for Android that allows users to play PS2 games on supported Android devices. This repository provides information about the application and its features.

> **Disclaimer:** This repository is not affiliated with or endorsed by the AetherSX2 developers or Sony Interactive Entertainment.

## Features

- High-performance PS2 emulation
- OpenGL, Vulkan, and Software rendering
- Save and Load States
- External Controller Support
- Touchscreen Controls
- Internal Resolution Scaling
- Per-Game Configuration
- Memory Card Management
- Fast Boot Support

## Requirements

- Android 8.0 or higher
- 64-bit ARM (arm64-v8a) device
- A legally obtained PlayStation 2 BIOS
- PS2 game backups that you own

## Installation

1. Download the latest APK from the Releases section.
2. Enable **Install Unknown Apps** in your Android settings if necessary.
3. Install the APK.
4. Launch the application.
5. Import your legally obtained PS2 BIOS.
6. Add your game library and start playing.

## How to Use

1. Open AetherSX2.
2. Configure the BIOS.
3. Scan your game directory.
4. Select a game.
5. Adjust graphics and controller settings if needed.
6. Enjoy your games.

## FAQ

### Does AetherSX2 include a BIOS?

No. You must dump the BIOS from your own PlayStation 2 console.

### Does it include games?

No. Users must provide their own legally owned game backups.

### Is internet required?

No. After installation and setup, most games can be played offline.
